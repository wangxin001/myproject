职能介绍 　　价格监测工作是政府价格主管部门对重要商品价格和服务价格以及相关生产流通环节的产量、成交量等的变动情况进行跟踪、采集、分析、预测和公布的宏观经济监测活动。价格监测中心的主要职责是：负责我市市场主要商品、服务及房地产价格及量的监测工作，实施价格监测报告制度，准确分析市场价格、供求变动趋势和经济发展形势；跟踪反馈国家重要经济在价格领域的反映，实施价格预测、预警，并及时提出政策建议，更好地为政府宏观调控和价格监管服务。主要工作是在日常数据监测的基础上，通过专题调查、调查问卷等方式进行补充，结合相关信息进行数据再加工，进行相关分析、预测，为市政府及其他政府部门提供各类分析报告，并为政策的制定提供重要参考依据。
